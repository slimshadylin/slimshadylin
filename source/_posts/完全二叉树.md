# 判断是否为完全二叉树


## 解题思路

1. 完全二叉树必须先有左孩子，再有右孩子，如果一个节点左孩子为空，右孩子不为空，那不是完全二叉树
2. 叶子结点：没有左孩子也没有右孩子
3. 层序遍历二叉树，如果遇到叶子结点，那么后续结点必须全都是叶子结点，否则不是完全二叉树

## 代码实现

``` java
import java.util.LinkedList;

public class IsCBT {
    
    public static class TreeNode {
        Integer value;
        TreeNode left;
        TreeNode right;
        
        public TreeNode(Integer value) {
            this.value = value;
        }
    }

    public static boolean isCBT(TreeNode head) {
        if (head.left == null && head.right == null) {
            return true;
        }
        boolean isLeaf = false;
        LinkedList<TreeNode> queue = new LinkedList<>();
        queue.add(head);
        while (!queue.isEmpty()) {
            TreeNode cur = queue.poll();
            // 当前结点的孩子有右无左，必不是完全二叉树
            // 出现过左右孩子不全的结点后，再次出现，也不是完全二叉树
            if ((isLeaf && (cur.left == null || cur.right == null)) || (cur.left == null && cur.right != null)) {
                return false;
            }
            if (cur.left != null) {
                queue.add(cur.left);
            }
            if (cur.right != null) {
                queue.add(cur.right);
            }
            if (cur.left == null || cur.right == null) {
                isLeaf = true;
            }
        }
        return true; 
    }
}
```
